# NEON BREACH — 설계 문서

> **V1 역사 문서:** V1 당시 승인된 10분 설계다. 현재 V2 구현 기준은 `2026-07-19-neon-breach-v2-product-contract.md`와 `2026-07-19-neon-breach-v2-story.md`다. 이 문서의 10분 런·아군 소멸·`내 군대` 표현을 V2에 적용하지 않는다.

날짜: 2026-07-15
상태: V1 당시 승인됨 (V2 구현 기준 아님)

## 1. 컨셉

사이버펑크 해커가 되어 몰려오는 보안 드론 군단을 상대로 10분간 생존하는
서바이버라이크(뱀파이어 서바이버즈류) 웹 게임. 고유 메커닉은 **적 해킹 → 아군화**.
적을 해킹해 내 편으로 전향시키며 화면을 내 군대로 채운다.

- 플랫폼: 웹 브라우저 (데스크톱 우선, 키보드 조작)
- 아트: 픽셀아트 × 네온 블룸 하이브리드 (Dead Cells / Katana ZERO 계열 감성)
- 참고작: 일기당천(threesur.midagedev.com), Vampire Survivors

## 2. 코어 루프

1. WASD / 방향키로 이동. 무기는 전부 자동 공격.
2. 적 처치 → 데이터 조각(XP) 드랍 → 주워서 레벨업.
3. 레벨업 시 선택지 3장 중 1장 선택 (새 무기 / 보유 무기 강화 / 패시브).
4. 시간 경과에 따라 적 웨이브 강화 → **10:00에 보스 등장** → 처치 시 승리.
5. HP 0 → 게임 오버 (생존 시간·처치 수 표시, 재시작 버튼).

## 3. 고유 메커닉: 해킹

- 적 처치 시 해킹 게이지 충전 (일반 적 1처치 = 5%, 엘리트 = 20%).
- 게이지 가득 → **Space** 로 가장 가까운 적을 해킹 → 아군으로 전향 (시안색 발광).
- 아군은 플레이어를 따라다니며 자동 전투. 최대 5기 유지 (초과 시 가장 오래된 아군 소멸).
- 엘리트를 해킹할수록 강한 아군 획득 → "누굴 해킹할까" 고르는 전략성.

## 4. 콘텐츠 (MVP)

### 무기 5종
| 무기 | 동작 |
|---|---|
| 펄스 SMG | 가장 가까운 적에게 자동 연사 |
| EMP 노바 | 주기적으로 플레이어 주변 광역 데미지 |
| 유도 미사일 | 무작위 적 추적 폭발 |
| 관통 레이저 | 직선 관통 빔 |
| 데이터 스파이크 | 플레이어 주위 궤도 회전체 |

각 무기 레벨 1~5 (데미지/발사수/범위 성장).

### 패시브 5종
이동속도 · 공격력 · 쿨다운 감소 · 획득 범위 · 최대 HP

### 적 4종 + 보스 1종
| 적 | 특성 |
|---|---|
| 러시봇 | 플레이어에게 돌진, 접촉 데미지 |
| 슈터봇 | 거리 유지하며 탄 발사 |
| 탱크봇 | 느리고 단단함 |
| 엘리트 | 강화 스탯, 해킹 가치 높음, 주기적 스폰 |
| 보스 | 10:00 등장, 패턴 2개: ① 예고선 후 돌진 ② 원형 탄막 발사 |

### 화면 흐름
타이틀 → 게임 → (레벨업 오버레이) → 승리 / 패배 → 재시작

## 5. 기술 구조

- **Vite + JavaScript + Three.js**
- 직교(orthographic) 카메라 탑다운. 게임 로직은 평면 좌표(x, y)로 처리, Three.js는 렌더링 전담.
- `UnrealBloomPass` 로 네온 글로우 — 발광 요소(탄환, 이펙트, 네온 액센트)에만 블룸.
- 스프라이트: **프로시저럴 픽셀아트** (16×16 기준, 오프스크린 캔버스에 코드로 생성 →
  텍스처화, NearestFilter로 픽셀 유지, 정수 배율 렌더링). 외부 에셋 없음.
- 걷기/공격 애니메이션은 2~3프레임.
- HUD·레벨업 UI는 DOM 오버레이 (타이머, XP바, HP바, 해킹 게이지, 처치 수).

### 모듈 구성
```
src/
  main.js        진입점, 게임 루프
  renderer.js    Three.js 씬/카메라/블룸 셋업
  sprites.js     프로시저럴 픽셀 스프라이트 생성
  input.js       키보드 입력
  player.js      플레이어 상태/이동
  enemies.js     적 정의/AI/스폰 풀
  allies.js      해킹된 아군 AI
  weapons.js     무기 5종 + 발사체
  pickups.js     XP 조각/아이템
  spawner.js     시간 기반 웨이브 테이블
  hacking.js     해킹 게이지/전향 처리
  upgrades.js    레벨업 선택지 생성/적용
  hud.js         DOM HUD/오버레이
```

## 6. 검증 기준

- 적 150마리 이상 동시 표시에서 60fps 유지 (오브젝트 풀링 사용)
- 10분 런 완주 가능: 레벨업 → 보스 → 승리 화면까지 끊김 없이 진행
- 해킹한 아군이 전투에 기여하는 것이 시각적으로 명확할 것
- 픽셀이 뭉개지지 않고 또렷하게 렌더링될 것
